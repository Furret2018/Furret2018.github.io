# VS注释与取消注释快捷键
注释： 先CTRL+K，然后CTRL+C
取消注释： 先CTRL+K，然后CTRL+U

选中需要注释的代码段，按Ctrl + K + C 即可快速注释；
选中需要取消注释的代码段，按Ctrl + K + U 即可取消注释。
